### 4.前端知识之http

* **常见状态码(重点)**

```
    1XX 信息性状态码，表示接受的请求正在处理
    2XX 成功状态码，表示请求正常处理完毕
    3XX 重定向状态码，表示需要客户端需要进行附加操作
    4XX 客户端错误状态码，表示服务器无法处理请求
    5XX 服务器错误状态码，表示服务器处理请求出错

    200 OK
    301 Moved Permanently
    304 Not Modified
    400 Bad Request
    403 Forbidden
    404 Not Found
    500 Internal Server Error
```

* http常见报文头
* cookie与session
* get/post的区别与联系
* 什么是跨域，以及如何处理(各种方式的优缺点)
